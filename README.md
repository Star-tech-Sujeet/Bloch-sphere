🌌 Quantum Gate Visualizer — Bloch Sphere

An interactive Bloch Sphere Visualizer built using Three.js, designed to help understand quantum states and quantum gates visually.

This project allows users to apply quantum gates and observe how a qubit evolves in real time on the Bloch sphere.

🚀 Features
🎯 Real-time Bloch Sphere visualization
🔄 Apply quantum gates interactively:
Pauli Gates: X, Y, Z
Clifford & Phase Gates: H, S, T, S†
📊 Live state representation:
Dirac notation (|ψ⟩)
Amplitudes (α, β)
Probabilities
Bloch coordinates (X, Y, Z)
🎮 Interactive controls:
Drag to rotate sphere
Scroll to zoom
📜 Gate history tracking
⚙️ Display toggles:
Axes
Grid lines
State labels
🔁 Quick state reset:
|0⟩, |+⟩, |−⟩
🧠 Concepts Covered

This project helps visualize:
